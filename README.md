## 🚀 快速开始

### 安装方式

 **开发者模式安装**
   - 下载本仓库代码
   - 打开 Chrome 扩展程序页面 (`chrome://extensions/`)
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择代码目录即可

### 使用说明

1. 打开新标签页即可使用
2. 点击左上角设置图标进行个性化配置
3. 复制网址并粘贴到页面任意位置添加网站
4. 拖拽图标可以调整顺序
5. 右键图标可以编辑或删除
6. 支持自定义背景设置，可调节模糊度与亮度
